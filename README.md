# AI Chatbot Multi-Model

A cross-platform AI chatbot application (Web + Mobile) with support for multiple LLMs including OpenAI, Azure Copilot, Google Gemini, Anthropic Claude, and xAI Grok.

## Features

- 🤖 Support for multiple LLM providers
- 💬 Real-time chat interface
- 💳 One-time payment processing with Stripe
- 📱 Mobile app (React Native - iOS & Android)
- 🌐 Web app (React)
- 💾 Chat history persistence
- 🔐 Secure authentication
- ⚡ Fast, responsive UI

## Tech Stack

**Backend**: Node.js + Express + PostgreSQL + Prisma
**Web**: React 18 + TypeScript + Tailwind CSS
**Mobile**: React Native + Expo
